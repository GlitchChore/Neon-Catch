# NEON CATCH 🌈🔫

**Verstecken, Fangen & Parkour im Neon-Look**

NEON CATCH ist ein Multiplayer Party-Game für Kids 6+ in Unity 6. 
0 Werbung. 0 Echtgeld während dem Spielen. Nur Spaß + Cosmetics.

---

### **5 Modi - 1 Game**
1.  **PAINT & FREEZE**: Mal dich an + versteck dich. 30 Sek Freeze.
2.  **NEON DOWN**: Free-For-All Shooter. 6 Schuss, 2 Sek Cooldown.
3.  **NORMAL TAG**: Klassisch Fangen. Letzter Überlebender gewinnt.
4.  **NEON HIDE**: Dunkle Map. Nur Taschenlampe vs. minimal leuchten.
5.  **NEON RUN**: Parkour Race ohne Fallschaden. Erster im Ziel.

### **10 Maps mit Parkour + Verstecke**
Sky City, Riesen Supermarkt, Fabrik, Hallenbad, U-Boot Wrack, Arkaden, Baustelle, Candy Fabrik, Wüsten Oase, Dachgarten.

### **Kids-Safe by Design**
- **Chat**: Nur Quick-Chat Buttons. Kein Tippen.
- **Namen**: Nur `Mantschgerl#1234`. Keine echten Namen.
- **Monetarisierung**: Nur Cosmetics. 0 Pay2Win. Tageslimit 300 Juwelen.
- **Server**: P2P über Unity Relay. 0€ Serverkosten.

### **Tech Stack**
- **Engine**: Unity 6.4 `6000.4.6f1`
- **Netcode**: Netcode for GameObjects + Unity Relay
- **Zielplattform**: PC + Android MVP
